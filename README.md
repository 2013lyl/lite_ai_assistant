# lite_ai_assistant
It can assist you in completing some computer file operations and also recognize text in image files

Not just a chatbot, but also your file manager. Without leaving the software, you can let AI help you organize documents and recognize text in images

# Key Features

Lite AI Assistant is more than a chatbot—it's your intelligent desktop companion that connects AI with your local files.

Intelligent Conversation: Supports leading LLMs for natural, contextual dialogue.

File Management: The assistant can read, create, delete, and rename​ files locally upon your request.

Image OCR: Upload images to automatically extract text via OCR, then discuss the content with the AI.
